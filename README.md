## Configuración de entorno

1. Crea `entornos/dev.json` con base en `entornos/dev.json.example`.
2. Ejecuta la app con:
   `flutter run --dart-define-from-file=entornos/dev.json`
3. Alternativa en PowerShell:
   `.\tool\ejecutar_dev.ps1`

Si `API_URL_BASE` no está definida, la app fallará al iniciar con un error claro.
